# Bridge-Maker Docs

This folder is the product documentation site for the current contract-first MVP.

Most library documentation sites are built with tools such as Docusaurus, VitePress,
MkDocs, or Next.js/MDX. For this repo we keep the docs dependency-free for now:

- `index.html` is a static landing/docs page.
- `styles.css` contains the site styling.
- Markdown files are source pages that can later be moved to Docusaurus, MkDocs,
  or VitePress without changing the content strategy.

Open locally:

```powershell
start docs\index.html
```

Current product line:

> Annotate existing gameplay semantics, export a contract, run automated QA, and
> receive bug evidence.
